# THREE.js project with npm

# after clonning the repo run npm install in the project root directory

to build run in command line: npm run build
project will be created in dist folder

_Commands run to create it_

npm install --save three
npm install --save jquery
npm install --save dat.gui
npm install --save math
npm install --save three-addons
npm install webpack

then:

- add webpack.config.js
- add in package.json script section with build
